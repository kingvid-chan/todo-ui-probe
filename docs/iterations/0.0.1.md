# Iteration 0.0.1 — Initial Todo UI

## Scope
纯前端待办清单网页：添加任务、勾选完成、删除。纯 UI 交互，无后端无数据库。

## Owner Input
- e000001: "实现一个纯前端的待办清单(todo list)网页，支持添加任务、勾选完成、删除，纯 UI 交互，无后端无数据库"

## Requirements
- 输入任务名点击添加后出现在列表中
- 点击勾选框标记完成（划线+变灰）
- 点击删除按钮移除任务
- 空列表时显示引导提示
- 纯前端单页 HTML/CSS/JS，无外部依赖
- 浏览器刷新后数据不持久化

## Tech Stack
纯前端 HTML + CSS + vanilla JS，无框架无构建工具

## Deployment
Nginx sub-path: `/projects/todo-ui-probe/`
